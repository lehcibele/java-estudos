# Variáveis 

- Uma variável é uma porção de memória (RAM) utilizada para armazenar dados durante a execução dos programas.
- Sintaxe:
```bash
tipo nome = valor inicial;
```
- Exemplos:
```bash
int idade = 25;
double altura = 1.68;
char sexo = 'F';
```

## Tipos primitivos em Java

- Tipos numéricos inteiros:
    - byte - 8 bits
    - short - 16 bits
    - int - 32 bits
    - long - 64 bits

- Tipos númericos com ponto flutuante:
    - float - 32 bits
    - double - 64 bits

- Um caractere Unicode:
    - char - 16 bits

- Valor verdade:
    - boolean - 1 bit

- String -> cadeia de caracteres (palavras ou textos)

- %f = Ponto flutuante
- %d = Inteiro
- %s = Texto
- %n = Quebra de linha
